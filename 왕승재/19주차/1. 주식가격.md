#### 1. 주식 가격 (Lv2)

```javascript
function solution(prices) {
    var answer = [];

    for (let i = 0; i < prices.length; i++) {
        let count = 0
        for (let j = i+1; j < prices.length; j++) {
            count += 1
            if (prices[i] > prices[j]) {
                break
            }
        }
        answer.push(count)
    }

    return answer;
}
```
