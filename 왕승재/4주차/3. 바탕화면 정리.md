#### 3. 바탕화면 정리 (Lv.1)

```python
def solution(wallpaper):
    x = []
    y = []
    
    for i in range(len(wallpaper)):
        for j in range(len(wallpaper[0])):
            if wallpaper[i][j] == "#":
                y.append(i)
                x.append(j)
    
    answer = [min(y), min(x), max(y)+1, max(x)+1]
    
    return answer
```
