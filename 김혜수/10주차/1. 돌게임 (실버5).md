## 돌게임 (백준 9655) 

[9655번: 돌 게임](https://www.acmicpc.net/problem/9655)

## idea

뭔가엄청난 계산을 해야할 것 같지만 결국 마지막에 이기는 사람을 구하는 게임이다. 3이나 1이나 사실 마지막 차례가 오는건 똑같다.

ex) 1인경우 → **상근**

3인경우 → 상근 - 창영 - **상근**

그래서 결국에는 짝수개가 있으면 창영이가, 홀수개가 있으면 상근이가 가져가게 된다.

```swift
let n = Int(readLine()!)!

// 3 = 1 + 1 + 1

if n % 2 == 0 {
    print("CY")
} else {
    print("SK")
}
```
